# UniFriends55

Готовый Telegram-бот для знакомств по интересам внутри университета.
Этот проект подготовлен для простого деплоя на **Railway.app**.

## Содержимое архива
- `bot.py` — основной код бота (Python, aiogram).
- `requirements.txt` — зависимости.
- `.gitignore` — файлы/папки, которые не нужно заливать в репозиторий.
- `README.md` — эта инструкция.

## Перед деплоем
1. Создай репозиторий на GitHub и загрузите туда файлы из архива.
2. Получи токен у @BotFather в Telegram (`/newbot`) — он будет вида `123456:ABC...`

## Деплой на Railway (быстро)
1. Зарегистрируйся на https://railway.app и подключись через GitHub.
2. Создай новый проект → "Deploy from GitHub repo" и выбери репозиторий с этим кодом.
3. В Railway перейди в **Variables** и добавь переменную:
   - `BOT_TOKEN` = `<твой токен от BotFather>`
4. В разделе Settings / Start Command укажи:
```
python bot.py
```
5. Нажми **Deploy**. Через минуту всё будет работать.

## Локальный запуск (для теста)
1. Установи Python 3.9+.
2. Создай виртуальное окружение:
   - `python -m venv venv`
   - `source venv/bin/activate` (Linux/macOS) или `venv\Scripts\activate` (Windows)
3. Установи зависимости:
   - `pip install -r requirements.txt`
4. Перед запуском выставь переменную окружения BOT_TOKEN:
   - Linux/macOS: `export BOT_TOKEN="123456:ABC..."`
   - Windows PowerShell: `$env:BOT_TOKEN="123456:ABC..."`
5. Запусти бота:
   - `python bot.py`

## Примечания и улучшения
- По умолчанию используется SQLite (`bot.db`). Для продакшена можно подключить PostgreSQL (потребуется адаптация).
- Фильтры можно расширить: по курсу, факультету, возрасту.
- Можно добавить модерацию и верификацию по e-mail университета.

Если хочешь — могу помочь залить репозиторий на GitHub и настроить Railway шаг-за-шагом.
